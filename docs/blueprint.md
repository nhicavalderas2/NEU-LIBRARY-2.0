# **App Name**: LibFlow NEU

## Core Features:

- Landing Page Navigation: Displays 'Welcome to NEU Library' with prominent buttons to navigate to Visitor Check-in, Admin Portal, or the 'Tap ID or Log In' option.
- Visitor Check-in & Logging: Provides an interface for visitors to log in via 'School ID Tap' or 'Institutional Email', select their 'Purpose of Visit', and securely records their entry into the library log system.
- Admin Portal Login & Authentication: A secure login interface allowing administrators to authenticate using username and password credentials to access the management dashboard.
- Admin Dashboard Metrics Display: Presents an overview of key statistics such as 'Today's Visitors', 'This Week', 'Blocked users', and 'Active Sessions', providing administrators with real-time insights based on logged data.
- Visitor Activity Log Reporting Tool: Renders a table displaying detailed visitor activity logs with columns for 'Time In', 'Name', 'College', 'Purpose', and 'Status'. Includes features for time period selection and the generation of PDF reports from historical data.
- Block List Management: An administrative interface for viewing, adding, or removing individuals from a block list, coupled with an info box explaining common reasons for a blocked entry.
- AI Visitor Welcome/Guidance Tool: A generative AI tool that dynamically provides contextual and helpful bullet points for the 'Why Log In?' footer section on the visitor check-in screen to inform and encourage users.

## Style Guidelines:

- Primary color: A deep, institutional blue (#235694), evoking trust and professionalism, used for main calls to action and significant UI elements.
- Background color: A very light, cool blue-grey (#F0F4F7), providing a clean and understated canvas for content, contrasting clearly with interactive elements.
- Accent color: A muted forest green (#2E4D3E), drawing inspiration from the user's request for a 'forest green theme', used for secondary interactive elements and branding details to add a natural, stable feel.
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, neutral, and highly readable qualities, ideal for both titles and detailed logs in an institutional application.
- Lucide-react for icons, selected for its modern, consistent, and extensive set of pictograms. Icons will focus on clarity and quick recognition for check-in, administration, reporting, and management tasks.
- A responsive design prioritizing adaptability across devices. Key layouts include a centered card approach for initial interactions (landing page, visitor check-in), and a structured multi-column dashboard with distinct areas for stats, logs, and management panels for administrators.
- Subtle and functional animations to provide intuitive user feedback, such as smooth transitions for form submissions, dynamic updates of dashboard statistics, and efficient navigation responses.
